
# Estatery

A real Estate “Rent a Property” web app with filtering mechanism.


## Tech Stack

React, Bootstrap



## Run Locally

Go to the project directory

```bash
  cd estatery
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

#### Server will be running on http://localhost:3000


## Features

- Responsive UI
- Multi Level Filtering
- clean Code
